| brgya la res ’ga’ gaṅ źig gis | | mi gsuṅ thugs las chuṅ dgyes pa || raṅ byuṅ bse ru lta bu
rnams | | rnam par ’gyiṅ źiṅ bźud gyur pa |

| ñams par ’gyur ba’i lam maṅ bas | | rtags par yaṅ ni rab dka’ ba || mya ṅan ’das pa’i lam
de ni | | khyod kyis lam chen srol du btod |

| de ni glaṅ po che rnams ltar | | dga’ mgur gźon nu dag daṅ ni || stobs daṅ blo ni sor gñis
pa’i | | bud med kyis kyaṅ goms par bgyis |

| de ni gzi byin ldan pa’i mchog | | khyod la bsten pa’i cho ’phrul te || saṅs rgyas kyi ni
ma ’dres pa | | mthu dpag med pa’i gzi byin lags |

| kun nas ñon moṅs sbrul dag gis | | rnam pa du mar zin rnams la || khyod ni spyi yi gñen po
yi | | skyon med ’byuṅ gnas ma ’dres pa |

| ’dod chags la sogs ’bags rnams la | | skyon ni skyon med mdzad khyod kyis || mi sdug sgom
pa la sogs pa’i | | chos kyaṅ rab tu bstan pa mdzad |

| khyod kyis lam gyi tshogs kyi chos | | dge ba zag bcas bstan mdzad pas || dṅos po rnams kyi
naṅ gi ni | | phan tshun khyad par dag kyaṅ bstan |

| de yaṅ ñe ba’i ñon moṅs daṅ | | ’bags phyir spyi yi rnam pas bcom || dur khrod mgal ba’i yog
pa ltar | | de dag ye śes me la bsregs |

| byaṅ chub pa yi sṅa rol du | | ’jug pa bde ba’i rgyu chos bstan || de nas byaṅ chub brñes
nas ni | | ldog pa bde ba’i chos kyaṅ bstan |

| saṅs rgyas nas kyaṅ dge ba’i chos | | zag bcas de dag ma bśad min || phal cher mya ṅan ’das
pa yi | | rgyu mthun ñid du yoṅs su bsṅos |

| khyod kyis gaṅ las ’gro ba dag | | ma draṅs rgud pa de gaṅ mchis || gaṅ dag ’jig rten ma sprad
pa’i | | phun tshogs de yaṅ gaṅ źig mchis |

| khyod kyis yid gñis mdud pa bcad | | lta ba ṅan pa’i tsher ma dkrugs || sred pa’i dug gi lcug
ma bsregs | | khro ba’i dug kyaṅ thams cad bcom |

| ma rig pa yi liṅ tog bsal | | ṅa rgyal gyi ni rgyal mtshan bsgyel || sems kyi tha ba rnams
kyaṅ bcom | | rnam par ’chiṅ ba rnam par gtubs |

| mdud pa thams cad rnam par bkrol | | chu bo la ni gziṅs kyaṅ bcas || khyod kyis chu bo rnams
kyi gźi | | mtsho chen gaṅ lags de yaṅ bskams |

| skyon daṅ sñigs ma yon po dag | | sems la mi mṅon ’khras bgyid pa || kun nas ñon moṅs dug
śiṅ gi | | rtsa ba khyod kyis druṅs nas phyuṅ |

| phyin ci log gis ’chal ba yi | | ṅan pa’i lam ni thams cad bcad || ’chi med bgrod pa gcig
pa yi | | dpal ldan lam ni draṅ po phye |

| gźan dag gdul bar bgyi ba yi | | skyon brgyad las ni ṅes draṅs nas || ’dod pa’i yan lag brgyad
ldan pa | | caṅ śes bzaṅ po’i gnas su bkod |

| kun nas ñon moṅs srin ’khrug pa’i | | mi gtsaṅ rgyun nas draṅs nas kyaṅ || ñon moṅs min pa’i
yan lag can | | dri med chu la ’phyo ru stsal |

| ’chal pa’i tshul khrims skyon bsal nas | | tshul khrims kyi ni byug pas byugs || ṅo tsha śes
daṅ khrel yod pa’i | | dri med gos kyis dgab pa mdzad |

| dam pa’i chos rgyan dkar po ni | | spa ba dag gis brgyan mdzad nas || de dag bsam gtan seṅ
ge’i khri | | chen po’i rkyen du ’bab la bźag |
