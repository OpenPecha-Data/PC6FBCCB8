| ljags yaṅs med par ’jig rten na | | kun mkhyen ñid ldan spobs pa ni || so so yaṅ dag rig pa
yi | | spobs pa’i khur ni su yis bkur |

| khyod kyis dgoṅs te de la ni | | chos ni gaṅ daṅ gaṅ bśad pa || ’di ni sñoms las mi mṅa’ bar
| | me tog bźin du phreṅ bar brgyud |

| gal te ’di yis ma bstan na | | khyod la kun mkhyen mṅar sus ’tshal || thugs rje chen po mṅa’
bar sus | | yon tan gźan yaṅ mṅar sus ’tshal |

| bstan pa dag ni ma mchis na | | khyod ’dra’i mkhas pa mṅon mi ’gyur || bstan pa yaṅ ni ljags
yaṅs la | | gźuṅ la dṅos grub bźin du brten |

| rnam pa sna tshogs chos gsuṅ źiṅ | | rnam pa sna tshogs ro bstar yaṅ || ’dam rdzab kyis ni
nam mkha’ bźin | | ro la chags pas gos mi mṅa’ |

| mi gsuṅ bźugs pa’i rjes su mthun | | gsuṅ ba la yaṅ skyo mi mṅa’ || mi mthun chos ni gñis
ka la | | gcig tu yaṅ ni mthun pa lags |

| mi gsuṅ bźugs na spro mi mdzad | | ston par mdzad la skyo mi mṅa’ || caṅ mkhyen ñid kyis dge
ba ste | | thams cad la ni sñoms par ’jug |

| ’di ltar thams cad gsuṅ spyod pa | | chos rnams ’ba’ źig bka’ stsol ba || spyod pa la ni ’jug
pa mtshar | | rigs pa la yaṅ gźol ba mtshar |

| ’byuṅ ba las gyur bsod nams daṅ | | chos gsuṅ ba yi raṅ bźin gyis || saṅs rgyas pa yi dri
źim pa | | gñis kyis rtag tu bsgos pa lags |

| mig gi spyod yul dag pa daṅ | | bsod nams sñan pa’i raṅ bźin gyis || śin tu rnam par grags
pa yi | | sñan pa gñis kyis phyogs su grags |

| gaṅ gis skyon kun med pa ’di | | źal nas phyuṅ ba mthoṅ gyur pa || skye bo de dag mig daṅ
ldan | | mig de dag kyaṅ dpal daṅ ldan |

| gaṅ gis de yi gsuṅ sñan pa | | mṅon sum rna ba’i sñim pa yis || bdud rtsi’i rgyun daṅ ’dra
’thuṅs pa | | mkhas pa de dag ches dpal ldan |

| sgra rnams brgyan par mdzad pa bźin | | sgra yis sa rnams brgyan pa bźin || bśad pa mdzad
na khyod kyi źal | | śin tu sba bar mdzad pa bźin |

| ’di ni ñid mdzes yon tan gyis | | thub pa khyod kyi zla źal ni || ’jig rten rgyan du gyur
pa yaṅ | | lhag pa’i rgyan du gyur par gda’ |

| mdog daṅ dbyibs daṅ sñan gsuṅ daṅ | | śin tu ’jam pa ñid daṅ mthus || ljags yaṅs lta bu gzugs
can gyi | | rdzas ni ’ga’ yaṅ ma mthoṅ ṅo |

| gal te lha mo dbyaṅs ldan ma | | de ni gzugs can ’ga’ mchis na || skyon kun med pa’i yan lag
can | | ljags yaṅs ’di ni de lags graṅ |

| yaṅs pa’i ljags kyis chos kyi rṅa | | kun tu rduṅ bar źugs pa’i tshe || lha daṅ mi yi sgra
rnams ni | | thams cad ro bro med par ’gyur |

| ston pa khyod daṅ ljags ’di daṅ | | bla na med pa’i bka’ ’di ni || ñon moṅs dug kun ’joms
pa yi | | yan lag gsum ldan sman gyi mchog |

| saṅs rgyas bcom ldan ’das la bstod pa bsṅags par ’os pa bsṅags pa las ljags la bstod pa źes
bya ba ste le’u bcu gcig pa’o |
