| saṅs rgyas chos ni bla med pa | | mi ’jigs pa daṅ stobs sogs kyis || ’gro ba’i naṅ du sgra
bsgrags pa | | dga’ bar ’os pa khyod la ’dud |

| ’phags chos dkar po bla med pa | | goms par bya ba rnams kyis ni || sku gñis po dag bsgos
gyur pa | | bsgom par bya ba khyod la ’dud |

| sku gsuṅ thugs kyi phrin las rnams | | śin tu rnam par byaṅ gyur pas || tshul khrims bsruṅ
ba las ’das pa | | gtsaṅ spra dag pa khyod la ’dud |

| ñon moṅs pa yi brjod pa spaṅs | | źi ba’i las kyi mtha’ la dgoṅs || mi gsuṅ daṅ du bźes mdzad
pa | | thub pa źi ba khyod la ’dud |

| bde ba daṅ ni sdug bsṅal gyis | | thugs mi ’phrogs la brtan pa’i thugs || thams cad zil gnon
sems can gyi | | sñiṅ po mchog gyur khyod la ’dud |

| kun la rnam pa thams cad daṅ | | rten gyi blo daṅ dgoṅs pa dag || legs par gnas phyir mi bsñel
ba’i | | chos can khyod la phyag ’tshal lo |

| ’jig rten ’das kyi yon tan phyug | | ’jig rten zaṅ ziṅ rnam par ’jig || ’jig rten snaṅ bar
mdzad pa po | | ’jig rten mgon po khyod la ’dud |

| bśes daṅ dgra daṅ bar mar sñoms | | sems can kun la phan dgoṅs pa || rjes su brtse bar mdzad
pa po | | thugs rje chen po khyod la ’dud |

| chos kyi ’du śes kun rdzogs pa | | skyob pa mṅon śes chen po mṅa’ || byas mkhyen ma byas mkhyen
pa po | | thams cad mkhyen pa khyod la ’dud |

| glags kyi skabs ni kun spaṅs śiṅ | | kun nas ’chiṅ ba bcad gyur pa || śes rab gsal bar mdzad
pa po | | sred pa bsal ba khyod la ’dud |

| mñes par bgyi ’os phyag bgyir ’os | | brgal bar ’os min noṅs mi mṅa’ || ’bad de mchod par
bgyi ’os pa | | rig pa gsum ldan khyod la ’dud |

| phan pa ma lags sel mkhas śiṅ | | phan pa bskyed par mṅon brtson pa || gcig tu brtse bar gźol
ba yi | | bśes gñen dam pa khyod la ’dud |

| skye dgu lam log źugs pa rnams | | rtag tu dam pa’i lam la ni || mṅon phyogs mdzad par mṅon
brtson pa | | rtsa lag dam pa khyod la ’dud |

| go ’phaṅ mkhyen pa rnams kyi phul | | khyod kyis go ’phaṅ mchog brñes nas || rkaṅ gñis rnams
kyi mchog gyur pa | | de las go ’phaṅ ’phags ma mchis |

| go ’phaṅ bla na med brñes nas | | sdug bsṅal thams cad sel mdzad ciṅ || ma chags ñer len mi
mṅa’ ba’i | | rkaṅ gñis mchog gyur khyod la ’dud |

| khyod kyi bstan la ma brten par | | ’gas kyaṅ de ñid mi mthoṅ bas || ’jig rten loṅ ba lta
bu yi | | mig tu gyur pa khyod la ’dud |

| sa steṅs gdugs ni ma mchis na | | źugs mchis mar me’aṅ mchis gyur la || rgyu skar nor bu zla
ba dag | | mchis kyaṅ gsal ba rdzogs mi ’gyur |

| snaṅ ba de daṅ gźan daṅ ni | | de bas khyad par ches ’phags pa || mchis kyaṅ khyod ni mi bźugs
na | | ’gro ba snaṅ ba mchis ma lags |

| de ltar ’jig rten ma rig pa’i | | mun nag thibs pa’i liṅ tog gis || bsgribs pa’i sems ldan
smag gyur pa’i | | sgron mar gyur pa khyod la ’dud |

| chos la blo gros ma byaṅ bas | | phan pa’aṅ phan par ma ’tshal źiṅ || gnod pa’aṅ gnod par
ma ’tshal la | | byaṅ ba’aṅ khyod la brten nas ’gyur |
