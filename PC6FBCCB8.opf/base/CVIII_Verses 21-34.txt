| byaṅ chub yan lag me tog ni | | zag med dri źim rnams kyis gtor || chos kyi dga’ ba ’phags
pa’i ros | | zaṅ ziṅ med pas tshim par mdzad |

| sred pa’i bran ni ’tshe maṅ ba | | thog ma med pa las bton nas || ñid kyi sras po gtan du
ni | | ñams pa med pa kho nar mdzad |

| gźan dag ma ’oṅs dus na ni | | raṅ rgyal ba yi snod du mdzad || gźan dag theg pa chen po yi
| | rgyal tshab mchog tu dbaṅ yaṅ bskur |

| chos kun gyi ni tshig mkhyen ciṅ | | don rnams kun gyi gźi la mkhas || skad kyi dbye ba kun
mkhyen ciṅ | | phyogs rnams kun tu spobs daṅ ldan |

| ṅa ni mgon med khyed rnams kyi | | grogs su gyur pa źes sgrogs śiṅ || thugs rje chen pos ’gro
ba ni | | mtha’ dag ’khyud pa bźin du bźugs |

| ston pa thugs rje che mṅa’ ba | | brtse bar ldan źiṅ thugs brtser mdzad || der brtson le lo
mi mṅa’ ba | | khyod daṅ ’dra ba gźan su mchis |

| khyod ni sems can kun gyi mgon | | spyi yi rtsa lag bzaṅ po lags || mgon du khyed bźugs ma
’tshal bas | | de slad skye bo byiṅ bar ᾽gyur |

| gaṅ źig yaṅ dag blaṅs nas ni | | tha śal bas kyaṅ grogs nus pa || grogs byed pa yi chos dag
ni | | khyod las gźan pa ’gas mi rtogs |

| phyi yi yan lag stobs kun ni | | khyod kyis yaṅ dag bsgrubs lags te || naṅ gi stobs dag ma
tshaṅ bas | | byis pa rnams ni sdug bsṅal sten |

| ’jig rten dpyod pa med daṅ sbyar | | chos ñid zab pa daṅ yaṅ sbyar || dman la mos pa ñid daṅ
sbyar | | lam ni ṅan pa maṅ daṅ sbyar |

| thog med dus nas goms pa yi | | brel ba ṅaṅ gis ’byuṅ daṅ sbyar || bsgom pa ’bad pa chen pos
kyaṅ | | thob par dka’ ba daṅ sbyar na |

| ’jig rten du ni khyod lta bu | | chos dkar yaṅ dag blaṅ ba yi || dṅos po gsal bar ston pa’i
rtsa | | bzaṅ po gal te ma byuṅ na |

| ’gro kun mun par ’gyur lags te | | mi ’dod tshor ba byuṅ ba yis || ñam thag ’o dod cher ’bod
pa’i | | ṅan soṅ raṅ bźin kho nar ’gyur |

| khyod la gaṅ gis ’gro rnam gsum | | de ni gźan gyis sgrub par bas || khyod ni dgra bśes bar
ma la | | bśes mchog ñid du brtse bar mdzad |

| saṅs rgyas bcom ldan ’das la bstod pa bsṅags par ’os pa bsṅags pa las phan par mdzad pa bstod
pa źes bya ba ste le’u brgyad pa’o |
