| chos rnams thams cad bdag med pa | | ’dus byas thams cad skad cig ma || mya ṅan ’das źi źes
bgyi ba | | mtshan ñid ’di gsum chos kyi rgya |

| smra ba kun gyi yul min pa | | mi phyed khyod kyis bźag pa mdzad || ji skad brjod kyaṅ ’jig
rten na | | raṅ bźin las ni gaṅ źig ’da’ |

| gal ’dzin pa yis śags la ni | | goms par bgyis kyaṅ su źig gis || me ñid graṅ ba lta bur ni
| | de ñid gźan du bgyi bar nus |

| nam mkha’ thogs pa med pa la | | ’ga’ yis rma phyin ’gyur srid kyi || chos kyi phyag rgya
’di la ni | | chos daṅ mthun par ’gal ma lags |

| chos ñid de ñid mkhyen khyod kyi | | bka’ la nag po chen po dag || bstan pas ’di ni mchis
grags kyi | | smra ba gźan la mchis ma lags |

| tha dad lta la brten pa yis | | ’jig rten rnams ni rtsod par ltuṅ || khyod ni gcig pus rtsod
med pa’i | | gnas kyi go ’phaṅ thugs su chud |

| loṅ ba rnams ni rtsod gyur kyaṅ | | mig can gzugs rnams mthoṅ ba ni || ji ltar rtsod par mi
’gyur ba | | de ñid khyod daṅ byis pa’i dpe |

| chos bźin gsuṅ ba khyod la ni | | ’ga’ daṅ lhan cig rtsod mi mṅa’ || legs par smra phyir mkhas
daṅ min | | khyad du gsod phyir mi mkhas min |

| ’dod pa rnams ni mi snaṅ gyur | | lta ba’i dra ba rab tu bcom || gźi ma mchis na ci slad khyod
| | gźi ma mchis pa᾽i rtsod par ’gyur |

| rgyal thabs brñes pa khyod lta bu’i | | brtan pa ci źig rtsod pa mdzad || ’od zer stoṅ ldan
ro lta bu’i | | mun daṅ lhan cig ci źig ’gyed |

| mkha’ la lcag gis ’ju ba bźin | | brtsad pas khyod la rdzi mi ’gyur || rluṅ gis bzlog phyogs
gtor ba’i rdul | | ’thor ba ñid la ’bab ciṅ ’go |

| ’jig rten rnams ni brtags par ltuṅ | | yaṅ dag ji bźin spyod yul khyod || de slad phyin ci
log gyur pa | | maṅ po rnams daṅ khyod mi mtshuṅs |

| dper na lcags daṅ gser bźin du | | log par mṅon par źen rnams daṅ || mṅon źen mi mṅa’ khyod
du ni | | lhan cig sgrun du ga la mchis |

| byis pa rnams ni ma ’tshal te | | mgon po phan daṅ bde bźed pa || do min khyod la bsdo bgyid
pa | | bdag gis bdag ñid brlag par bas |

| bdag gis bdag ñid me ’bar ba’i | | phreṅ ba ’khrigs par mi bsdo yi || rgyal ba thub pa med
pa yi | | gzi byin can la bsdor mi ruṅ |

| khyod la ’dir brgal dus btab nas | | khyod kyis slar gźan dus btab pa || brtod pa dam pos
btags pa bźin | | stan las ldaṅ bar nus ma mchis |

| khyod kyis tshogs par bden smra’i bu’i | | rgol ba’i mche ba bcag pas na || pha rol rgol ba
thams cad kyi | | rva yaṅ bcag pa ñid du ’gyur |

| pha rol rgol ba kha cig ni | | khyod daṅ phrad nas ’dud par bgyid || kha cig thag riṅ slar
ldog ste | | gźan dag spyan sṅar źum nas mchis |

| kha cig khyod kyis dam pa’i chos | | bdud rtsi’i rgyun gyi char phab pas || yaṅ dag sgrub
pa phun sum gyis | | dam pa’i chos la brten par bgyid |

| kye ma mgon po thugs rje’i bdag | | kyi hud gti mug ñon moṅs ṅan || kyi hud sred pa rnam par
’phro | | kyi hud ṅa rgyal phuṅ bar byed |
