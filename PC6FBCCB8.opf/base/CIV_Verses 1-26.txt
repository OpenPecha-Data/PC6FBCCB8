| bde bar gśegs pa khyod gcig pu | | chos gcig chos kun snaṅ mdzad pa || rnam pa thams cad mkhyen
pa yis | | de daṅ ldan gyi gźan ma lags |

| khyod ’ba’ źig gi ye śes kyis | | śes bya mtha’ dag khyab lags kyi || khyod las gźan pa thams
cad kyi | | śes bya kho na lhag pa lags |

| rgyu rkyen ’di las ’bras bu ’di | | ’di las ’di ni ma yin źes || gnas daṅ gnas min ṅes pa
kun | | khyod kyis mkhyen gyi gźan gyis min |

| las chos yaṅ dag blaṅs pa yi | | rnam smin dus ni thams cad pa || sa kun rtogs pa thams cad
ni | | rnam kun khyod mkhyen gźan gyis min |

| rnal ’byor sa ni ma lus pa | | kun nas ñon moṅs rnam gźag daṅ || rnam par byaṅ bcas zab mor
bcas | | khyod gcig kho nas thugs su chud |

| dbaṅ po mchog daṅ mchog min pa | | chuṅ daṅ ’briṅ daṅ chen po daṅ || cha śas dbye ba mtha’
yas pa | | khyod las gźan gyis rtogs ma mchis |

| mthun par ’gyur ba gaṅ lags daṅ | | mi mthun ’gyur ba gaṅ lags pa || mi yi mos pa sna tshogs
kun | | khyod gcig kho nas thugs su chud |

| ’jig rten ’di khams sna tshogs te | | sna tshogs khams ñid mtha’ dag ni || sna tshogs khams
dbye khyod gcig pus | | mkhyen gyi gźan gyis ma lags so |

| rgyu daṅ rkyen ni sna tshogs las | | sna tshogs mṅon par ’grub pa ni || kun du ’gro ba’i lam
stobs kyis | | thams cad khyod kyis thugs su chud |

| thog med srid par byuṅ ba daṅ | | tha ma med par ’byuṅ ’gyur gaṅ || de la khyod kyi ye śes
ni | | mṅon sum bźin du gzigs pa ’jug |

| sṅon gyi mtha’ daṅ phyi ma’i mthar | | ’jig rten yaṅ dag ’grub pa daṅ || de bźin ñiṅ mtshams
sbyor ba ni | | mtha’ yi bar bźin khyod kyis gzigs |

| ñe’am yaṅ na riṅ yaṅ ruṅ | | ji ltar khyod la mṅon sum pa || de ltar sgrib pa bag chags daṅ
| | bcas te khyod kyis spaṅs pa lags |

| dṅos ’di yoṅs su ma mkhyen ciṅ | | ñon moṅs ’di yi bag chags ni || lhag ma lus śes mi bsñad
pa’i | | go ’phaṅ de ni khyod kyis brñes |

| khyod kyis bar du gcod gsuṅs gaṅ | | ṅes par de dag bar du gcod || khyod kyis lam bstan gaṅ
lags pa | | de gcig kho na ṅes par ’byin |

| de yis bar gcod mi nus śe’am | | de yis ṅes ’byin mi ’gyur źes || gźan gyi klan ka cha śas
kyaṅ | | de la srid pa ma mchis so |

| khyod kyis rgyal bgyi ma rgyal min | | śes bya mi snaṅ bgyis ma mchis || dbaṅ phyug khyod
ñid kun mkhyen ciṅ | | kun rgyal lags kyi gźan ma lags |

| stobs ldan ’jigs mi ’gyur bas khyod | | mi bsñeṅs gaṅ lags rigs lags na || mthu mṅa’ bźin
du ’ga’ la yaṅ | | khyad du mi gsod gaṅ de rmad |

| log pa’i ṅa rgyal can ’ga’ źig | | dregs daṅ ’bral bar mdzad pa gaṅ || gdul ba yi ni thabs
tsam ste | | des ni stobs sñems yod ma lags |

| ’jig rten dag na stobs ldan pas | | kun las rgyal źes gaṅ brjod pa || de ni gźan la bstod
par bas | | go ’phaṅ de ni khyod la rdzogs |

| ’dren pa khyod la chos mthun par | | ’gas kyaṅ brgal bar nus ma lags || klan ka btsal ’os
ma lags te | | noṅs mi mṅa’ ba’i bde ba brñes |

| noṅs mi mṅa’ bas gaṅ las kyaṅ | | bźi ni bsruṅ bar bgyi mi ’tshal || bźi po yoṅs su rdzogs
slad du | | khyod la sus kyaṅ glags mi thod |

| kun la mi thogs blo gros can | | dgoṅs pa kun tu ñe bar gnas || kun la thugs rje sñoms pa
po | | thugs ni kun la chags mi mṅa’ |

| śes bya’i rgya mtsho kun rgal ba | | bag chags thams cad gtan nas spaṅs || bar chad kun gyi
khyad par mkhyen | | ṅes par ’byin pa kun la mkhas |

| kun gyi bsam pa mkhyen pa po | | gdul ba thabs kyi pha rol gśegs || ñon moṅs sna tshogs thams
cad kyi | | gñen po la yaṅ bsñeṅs mi mṅa’ |

| khyod la brten nas smra ba po | | byis pa śed bye ma gyur gaṅ || de ni ma rig mun pa yi |
| tshaṅ tshiṅ mi bzad mthu lags so |

| khyod kyi bka’ la brten nas kyaṅ | | bral bar ma gyur gaṅ lags pa || de daṅ de sgrib ces bgyi
ste | | de daṅ de sgrib mi zad lags |

| saṅs rgyas bcom ldan ’das la bstod pa bsṅags par ’os pa bsṅags pa las stobs daṅ mi ’jigs pa
la bstod pa źes bya ba ste le’u bźi pa’o |
