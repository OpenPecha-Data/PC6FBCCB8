| ji ltar thub pa khyod kyi bka’ | | ñes dmigs brjod ldan thos pa ltar || ’khor ba’i sdug bsṅal
mi bzad pa’i | | me yis kun tu sbar kyaṅ ni |

| ’gro ba mi śes go bgos pa | | de ltar skyo bar ’gyur ma lags || de ltar mgon po khyod kyi
bka’ | | mṅon sum pas kyaṅ don gsal rab |

| chos ñid zab mo gtiṅ yaṅs pa | | bśad pas sa ler gyur lags pas || khyod kyi bka’ ni snrel
źi yi | | cho ’phrul dag kyaṅ mdzad pa lags |

| tshig gi mtha’ ni mi mthoṅ źiṅ | | don gyi gtiṅ yaṅ mi dpogs la || phan tshun rjes su źugs
pa ni | | de las ya mtshan che ci mchis |

| smra ba’i dam pa khyod kyi bka’ | | rnam graṅs du ma ñid mchis śiṅ || zlos bu yaṅ ni ma mchis
la | | sbyor ba’i lugs kyaṅ rigs pa lags |

| smra ba’i dam pa khyod kyi bka’ | | gnas pa bźi yi yon tan ldan || luṅ bstan bźi yi rten gyur
kyaṅ | | don du gñis su ’gyur ma mchis |

| gnaṅ ba daṅ ni dgag pa la | | khyod bka’ la lar gcig tu ṅes || la lar ṅes pa ma lags la |
| phan tshun ’gal ba’aṅ ma mchis so |

| chos kyi de ñid don slad du | | rtsod pa med pa’i go ’phaṅ gźi || rigs ’thad yon tan bzaṅ
po yis | | ṅan smra thams cad sun ’byin pa |

| gcig tu don dam la gźol la | | kun rdzob la yaṅ skyon ma mchis || bden pa gñis daṅ mi ’gal
bas | | khyod kyi bka’ ni rigs pa ’byuṅ |

| ñon moṅs med la gcig tu ’jam | | ñon moṅs can la ṅes brlaṅ yaṅ || khyod kyi bka’ ni bsñen
pa daṅ | | bsriṅ ba’i skyon daṅ ldan ma lags |

| rnam par byaṅ ba’i rjes mthun la | | de yi phyogs su lhuṅ mi mṅa’ || kun nas ñon moṅs dgrar
gyur la | | mi bzod mi mṅa’ khyod kyi bka’ |

| smod pa’i tshig ni brgyas gaṅ yaṅ | | phyar ka btoṅ ba’i skyon ldan min || rab tu bsṅags pa’i
tshig phyug kyaṅ | | ’ga’ la’aṅ rten pa mṅa’ ma lags |

| bsnan daṅ dor ba mi mṅa’ la | | bsdus daṅ rgyas pa daṅ yaṅ ldan || brtags śiṅ ’thad par ’gyur
lags la | | daṅ po nas kyaṅ mi bzaṅ min |

| śin tu sñan gsuṅ chags mi ’gyur | | maṅ la sñiṅ po ma mchis min || sna tshogs pa la don ma
’khrugs | | bka’ ni ṅo mtshar maṅ po lags |

| gsuṅ gi bdag po khyod kyi bka’ | | yid ’phrog mdzad la źen mi mṅa’ || gzi byin can la gnod
mi ’gyur | | khyab la ṅan pas rtogs ma lags |
