| bgo bśa’ mdzad pa’i chos ñid ni | | khyod gcig pu la rdzogs mṅa’ yi || gźan la phan gdags
bgyi ba la | | skye bo gźan ni tshogs śiṅ mchis |

| khyod kyi slob ma bsdu ba yi | | dṅos la mkhas par gda’ ba gaṅ || de yaṅ rdzogs par mi mkhas
pas | | khyod kyi cha śas rjes su ’braṅ |

| bcom ldan khyod kyi zaṅ ziṅ gi | | bgo bśa’ srog gi bar lags te || dam pa’i chos kyi bgo bśa’
ni | | slob dpon dpe mkhyud bral ba lags |

| khyod ni bgo bśa’ bgyi ba’i las | | bla na med pa’i mdzad pa po || phun sum thams cad rdzogs
mṅa’ ba | | khyod ni btaṅ bar bgyi ma lags |

| ’gro ba phal cher mi bzad pa | | gźan la gnod pa bgyid par dga’ || khyod ni ’jig rten raṅ
bźin te | | rgyun las bzlog pa’i phyogs su ’jug |

| byas pa gzo gaṅ de dag ñuṅ | | phan ’dogs bgyi ba ’ga’ tsam ste || de yaṅ phal cher re ba
can | | khyod gcig dag pa’i chos can lags |

| phan btags pa yis khyod mi ’jug | | gnod pa bgyis pas ldog ma lags || skal ba mchis śes thams
cad la | | khyod ni mñam pa ñid du ’jug |

| mtshan mo me tog ku mu da | | kha ’bye ’gyur źiṅ padma zum || ñin par padma kha ’bye źiṅ |
| cig śos zum par ’gyur mod kyi |

| ’on kyaṅ ñi ma zla ba yi | | ’od la phyogs su lhuṅ ma mchis || de ñid khyod kyis byis pa daṅ
| | mkhas pa dag la bstan pa’i dpe |

| gaṅ tshe mgon khyod mdzad pa ni | | thams cad gźan don kho nar bas || dpag med phan pa mdzad
pa po | | khyod la lan cis lon par ’gyur |

| sems can dar bar mi ’gyur ba’i | | sgrub pa khyod la ’ga’ mi mṅa’ || khyod kyi dbugs phyuṅ
tsam yaṅ ni | | ’gro la phan daṅ bde ba skyed |

| saṅs rgyas dbugs ni phyuṅ ba’i ṅad | | dri kun zil gyis gnon mdzad pa || tshor na źe ’gras
gyur pa yaṅ | | khyod la daṅ ba bskyed par bgyid |

| saṅs rgyas dbugs kyi ṅad bab na | | bag med pa yi lha rnams kyaṅ || lha yi yul yaṅ spaṅs nas
ni | | ’dul ba dag la mṅon phyogs ’gyur |

| khyod las dam pa’i chos mñan pas | | thams cad tshim par ’gyur ma ’tshal || de slad dam pa’i
don du na | | khyod ñid ṅoms ’gyur tshim mdzad lags |

| skye dgu rnams la char bźin du | | khyod ni mkhas pa kun gyi gsos || ’di ltar khyod kyi gsos
’gyur gaṅ | | de daṅ de ni mchis ma lags |

| dgroṅs par ’tshal ba rnams la yaṅ | | khyod kyi phan dgoṅs ci ’dra ba || de blo ma sbyaṅs
byis pa yis | | yid ches bgyi bar nus ma lags |

| khyod ni sdaṅ ba rnams la yaṅ | | legs pa ci ’dra sgrub mdzad pa || sa de yid la smon pa’i
yaṅ | | yul las ’das par gyur pa lags |

| la la phan ’dogs mdzad pa yi | | ro mchog rtogs pa de dag kyaṅ || khyod kyi chos kyi bgo skal
la | | spyod par ’gyur ba’i gñen lags so |

| gal te sems can ’di dag gis | | khyod mṅa’ thugs rje’i dgoṅs pa ’tshal || khyod kyi bstan
pas ’tsho bźin du | | kun tu ’bar ba’i mer yaṅ ’jug |

| gaṅ slad bdud kyi dgras thebs pas | | rtogs par mi ’gyur de slad du || khyod kyi bstan las
’gal bgyid pa’i | | byis pa rnams ni bdag ñid ’gums |
