| dam chos ma yin ’di gsum gyis | | skye bo ’di ni kun dkrugs te || mi bzad rgyun gyis ’das
pa bźin | | khyod kyi gźuṅ daṅ bral bar bgyid |

| ’di yi gźi yis lta ba daṅ | | the tshom la sogs ñe ñon moṅs || so sor lam gyi bar gcod pa
| | kun nas ñon moṅs thams cad ’phel |

| ka la biṅ ka’i skad thos nas | | khva yi sgra la chags med ltar || khyod kyi chos kyi ro myoṅ
na | | gźan gyi tshig la chags mi ’gyur |

| gaṅ dag chos la sgyu bśams nas | | khyod kyi bka’ la źugs gyur pa || ṅan ṅon thos pa de la’aṅ
de | | rṅa mo sbraṅ rtsi bźin du ’gyur |

| bden źiṅ go sla don daṅ ldan | | bsam pa daṅ ni cho ’phrul bcas || zad mi śes la mdor ’dus
pa | | gcig tu mthu ni chen por ldan |

| sñiṅ po mchog ldan ro mchog ldan | | rnam graṅs ldan źiṅ thabs daṅ ldan || yoṅs su rdzogs
la ma ’dres pa | | gzi brjid daṅ ldan grags daṅ ldan |

| de ’dra’i yon tan daṅ ldan pa’i | | khyod gsuṅ thos pa su źig ni || de las bzlog pa gźan gyi
tshig | | rul pa lta bu snom par bgyid |

| sdug bsṅal yaṅ ruṅ bde yaṅ ruṅ | | don daṅ mi ldan gaṅ yin pa || de ni yoṅs su spaṅ bya źiṅ
| | bzlog pa bsten par bya bar gsuṅs |

| ’dod daṅ mi ’dod bsten pa la | | de ltar phyad par gsuṅs khyod kyi || lam mal thaṅ chad smra
ba rnams | | ji lta bur na rjes ’braṅ ’gyur |

| spyad pa rnams la khyod kyis ni | | ’di yi ched du bstan pa gaṅ || de ñid khyod kyi bka’ daṅ
ni | | gźan gyi tshig gi mtshams rdzogs pa |

| khyod kyi bka’ daṅ gźan gyi tshig | | ji ltar khyad par che źugs pa || dug daṅ bdud rtsi de
lta min | | mun daṅ snaṅ ba’aṅ ma lags so |

| khyod kyi bka’ daṅ gźan gyi tshig | | ji ltar khyad par rgyaṅ chad pa || mtho ris ṅan soṅ
de ltar min | | zil dṅar bkus te bor ba’aṅ min |

| de ni tshig tsam sñiṅ po ste | | ’di ni tshig daṅ tshig don ldan || de yis khyod kyi bka’
daṅ ni | | gźan gyi tshig gi khyad par bśad |

| bde bar gśegs pa khyod daṅ ni | | smra de rnams kyi khyad par gaṅ || de ñid khyod kyi bka’
daṅ ni | | gźan gyi tshig gi mtho dman lags |

| ji ltar srid daṅ thar pa’i khyad | | de ni dpyad du med pa ltar || dpa’ bo khyod kyi bka’
daṅ ni | | gźan tshig khyad kyaṅ de daṅ ’dra |

| ’jug daṅ ldog pa gaṅ lags daṅ | | kun nas ñon moṅs rnam byaṅ gaṅ || de ñid dpa’ bo khyod bka’
daṅ | | gźan gyi tshig gi khyad par lags |

| ’di ni gcig pu de bźin ñid | | de ni bslu chos ’ba’ źig na || khyod kyi bka’ daṅ gźan gyi
tshig | | de las khyad źugs gźan ci ’tshal |

| ’di ni gcig pu dge la de | | gcig tu bar chad bgyid lags na || khyod kyi bka’ daṅ gźan gyi
tshig | | de las mi ’dra ci źig mchis |

| de yis bsgos śiṅ ñer bsgos pa | | ’di yis dag par bgyid ’gyur ba || de ñid mgon po khyod bka’
daṅ | | gźan gyi tshig gi khyad par lags |

| de kun dam pa ma yin brdol | | ’di gcig legs gsuṅs de yi slad || khyod kyi bka’ daṅ gźan gyi
tshig | | brtsan daṅ mi brtsan ṅes pa lags |

| saṅs rgyas bcom ldan ’das la bstod pa bsṅags par ’os pa bsṅags pa las rtsod pa mi mṅa’ bar
bstod pa źes bya ba ste le’u drug pa’o |
