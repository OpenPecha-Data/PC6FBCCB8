| ñi ma’i gñen gyur ñi ma yi | | ’od pas śin tu ’od gsal ba || phyag bya kun gyi rab daṅ mchog
| | phyag ’os phyag bgyis khyod la ’dud |

| stobs chen dpa’ bo chen po pa | | mṅon śes che la mthu yaṅ che || g᾽yul ṅor mda’ chen ’phen
mdzad pa | | dga’ ston chen po khyod la mchod |

| ’jig rten mchod ’os mchod ’os mchog | | ’jig rten mchod pas mchod pa po || mchod pa’i snod
ni bla med pa | | mchod par ’os pa khyod la mchod |

| mchod daṅ mkhyen pas rtogs pa brñes | | dgoṅs pa ṅa rgyal bag mi mṅa’ || brjed ’os khyod la
ṅa rgyal gyi | | sems bor nas ni brjed par bgyi |

| gus bgyid ma gus pa la sñoms | | ’jig rten gus pas mchod bgyis pa || khyod la bkur stir gźol
ba yi | | dbaṅ pos gus par bkur sti bgyid |

| khyod daṅ ’dra ba ’ga’ ma mchis | | lhag pa ma byuṅ ’byuṅ mi ’gyur || de slad sems can kun
la khyod | | dgra zin sñoms pa’i mñam med lags |

| ma ’dres ’phags pa’i gduṅ las byuṅ | | saṅs rgyas kun daṅ skal pa mñam || mdog daṅ śes rab
snaṅ ba yis | | phyogs rnams snaṅ bar mdzad de ’khruṅs |

| sum cu rtsa gñis mtshan daṅ ldan | | mdom gaṅ ’od kyi ’od zer spro || mtshan daṅ mthun pa’i
dpe byad bzaṅ | | brgyad cus ’bar ba’i gzi brjid can |

| rgya chen dpal daṅ ldan pa’i sku | | kun gyi mdzes pa zil gnon pa || ’jig rten snaṅ bar mdzad
nas ni | | ñi ma gźan źig lta bur bźugs |

| bcom ldan dus kun rtogs pa’i chos | | rnam kun ’byuṅ gnas thams cad ni || phyag mthil skyu
ru ra lta bur | | khyod kyi thugs ni spyod yul gyur |

| chos rnams g᾽yo daṅ mi g᾽yor bcas | | gcig daṅ sna tshogs so so la || mkha’ la rluṅ dag rgyu
ba bźin | | khyod kyi thugs ni thogs mi mṅa’ |

| bya ba mdzad pa khyod kyis ni | | ji ltar thugs bskyed tsam gyis su || khyod la bźed dgur
kun ’grub pa | | de ltar chos dbyiṅs legs thugs chud |

| dge mthar phyin pa khyod la ni | | sbyor byuṅ dge ba ’ga’ mi mṅa’ || khyod ni gaṅ źig bźed
na yaṅ | | bźed pa tsam la rag las so |

| chos rnams kyi ni chos yod daṅ | | chos ñid daṅ ni chos kyi mthu || so sor rgyu daṅ bcas gnas
śiṅ | | rkyen daṅ ’bras bur bcas pa daṅ |

| khams daṅ dbye ba tha dad bcas | | skye daṅ ’jig bcas rgyas pa de || thams cad so sor rnam
kun tu | | kun mkhyen khyod kyis mkhyen gźan min |

| khyod la śin tu bsgribs pa’aṅ mṅon | | śin tu riṅ ba’aṅ ñe ba ste || śin tu ’dziṅs pa’aṅ sal
gyis dag | | śin tu mthon po’aṅ dma’ ba lags |

| śin tu gnag pa’aṅ snaṅ bar bcas | | śin tu the tshom gnas kyaṅ ṅes || śes bya śin tu zab mo
yaṅ | | khyod la sa ler gda’ ba lags |

| dbaṅ po chen po’i glaṅ chen gyi | | rked bciṅs gser las bgyis pa ltar || chos ñid śin tu zab
mos kyaṅ | | khyod ni ’o brgyal ’gyur ma lags |

| chos kyi naṅ na chags bral ltar | | ṅes ’byin naṅ na draṅ po ltar || phun tshogs naṅ na saṅs
rgyas ltar | | khyod ni sems can kun gyi mchog |

| rnam grol naṅ na mi g᾽yo ltar | | rigs pa’i naṅ na chos ñid ltar || śes rab naṅ na kun mkhyen
ltar | | khyod ni lus can kun gyi mchog |

| dag pa’i naṅ na rnam grol ltar | | de ñid naṅ na stoṅ pa ltar || dka’ thub naṅ na bzod pa
ltar | | khyod ni srog chags kun gyi mchog |

| gzugs kyi naṅ na khyod sku ltar | | sems kyi naṅ na khyod thugs ltar || chos kyi naṅ na khyod
chos ltar | | khyod ni srog chags kun gyi mchog |

| saṅs rgyas bcom ldan ’das la bstod pa bsṅags par ’os pa bsṅags pa las | thams cad mkhyen pa
ñid grub pa źes bya ba ste le’u gsum pa’o |
