| ’jug pa daṅ ni ldog pa la | | ’gro ’di’i raṅ bźin gaṅ lags daṅ || lha daṅ bcas pa’i ’jig rten
kun | | so sor gaṅ las mi ’da’ ba’i |

| lugs mthun lugs daṅ mi mthun pa | | tshig bcu gñis kyi yi ge gñis || de ni dam chos ñi ma
yi | | ñi ma mchod pa’i ’os su bstan |

| de ñid rdzus te skye ba yi | | srid gcig yan lag ma tshaṅ bas || ji bźin yan lag so so la
| | bźi ru bsgres pas gsum mchod lags |

| de nid yan lag tshaṅ ba las | | yi ge bcu gñis tshig dag gis || bden pa’i lugs su sṅa ma bźin
| | bsgres na rab tu ’gro ba lags |

| tshaṅs pa ’di ni khyod bstan la | | gaṅ dag ji bźin bsten bgyid pa || bram ze de dag tshaṅs
’gyur ba’i | | gsaṅ gsum mthun pa’i tshaṅs pa lags |

| tshaṅs pa’i tshul daṅ tshaṅs pa daṅ | | tshaṅs tshul ’bras bu tshaṅs pa’i don || tshaṅs mchog
khyod kyi bstan pa las | | gźan la mchis pa ma lags so |

| tshaṅs pa’i tshul ni ma mchis par | | ’ga’ yaṅ bram zer ’gyur ma lags || draṅ po yi ni lam
’di yaṅ | | gźan ni ’ga’ la’aṅ ma mchis so |

| rig daṅ źabs su ldan pa ni | | tshaṅs pa lags par bram zes bśad || rig pa de daṅ źabs de yaṅ
| | khyod kyi bstan las gźan la min |

| slob daṅ mi slob rig pa de | | źabs ni de thob bgyid pa lags || gźan la chos kyi lam de ni
| | miṅ yaṅ mchis pa ma lags so |

| gaṅ źig goms par bgyis pa yis | | rigs kun bram zer ’gyur bgyid pa || bdud rtsi’i sgo ni riṅ
źig nas | | bcad pa khyod kyis dbye ba mdzad |

| lta ba ṅan daṅ yid gñis kyis | | zin pa bram zer ci źig ’gyur || khyod kyi gźuṅ lugs ma gtogs
par | | gźan gyis mdud pa ’di mi chod |

| bram ze tshaṅs pa’i bu yin te | | sñiṅ daṅ kha las skyes źes pa || ’jig rten gyi ni grags
pa de | | khyod la ’bras bu bcas par gyur |

| tshaṅs khyod tshaṅs lam khyod la mṅa’ | | khyod ni bram ze’i khyu mchog ste || ñe bar ’dren
daṅ ston pa’aṅ khyod | | khyod ni dus mchod mdun na ’don |

| khyod kyis legs par bstan pa yi | | bram ze mi gtsaṅ ’gyur ma lags || ñe bar draṅs pa’i bram
ze yaṅ | | tshaṅs pa’i tshul las ltuṅ mi ’gyur |

| tshaṅs pa’i bden pa gsum daṅ ni | | tshaṅs pa’i mtshan ñid rnam gsum daṅ || tshaṅs pa’i mchog
rab gsum daṅ ni | | tshaṅs pa’i rigs kyaṅ mchis ma lags |

| khyod bstan las gźan ’ga’ la yaṅ | | bram zer ’gyur ba’i chos ma mchis || tshaṅs pa yi ni
mchod pa daṅ | | tshaṅs pa’i lam yaṅ mchis ma lags |

| ’jig rten legs bśad ji sñed pa | | thams cad khyod kyi bstan las ’byuṅ || skyon med mṅon par
brjod pa ’di | | bde gśegs kho na’i ma ’dres pa |

| khyod las gźan pa gaṅ dag cig | | ṅag las skyon med gda’ ba de || thams cad chos kun mkhyen
khyod kyi | | legs par gsuṅs pa’i bram ze lags |

| srin bus zos pa’i yi ge ltar | | ’di las gźan pa’i legs bśad gaṅ || de ni lcags la gser bźin
du | | de la ’tsham pa ma lags so |

| e raṇ ḍa yi śiṅ khrod du | | tsan dan sa mchog bźag pa ltar || ltag ’og ’gal ba de la de |
| mchis par su źig yid ches ’gyur |

| gaṅ źig nim pa’i śiṅ la ni | | a mra’i ’bras bu smin btags pa || de las skyes śes yid ches
te | | gźan la legs bśad yod sems graṅ |

| ma la ya las tsan dan bźin | | khyod las gźan la lhar bcas pa’i || ’jig rten ’di na legs bśad
pa’i | | rin chen ’byuṅ ba ma mchis so |

| saṅs rgyas bcom ldan ’das la bstod pa bsṅags par ’os pa bsṅags pa las tshaṅs pa daṅ mthun
par gsuṅ bar bstod pa źes bya ba ste le’u bdun pa’o |
