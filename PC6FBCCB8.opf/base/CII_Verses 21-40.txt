dpon bya ba mdzad | | ’phags pa khyod la phyag ’tshal lo |

| thugs ni dka’ thub brtul źugs dag | | yaṅ dag blaṅs pas gtsaṅ mar mdzad || sdig pa thams cad
bkrus gyur pa | | khrus mdzad khyod la phyag ’tshal lo |

| mṅa’ bdag mchog ni brñes gyur ciṅ | | sems can thams cad zil gyis gnon || seṅ ge’i ṅa ro sgrogs
mdzad pa | | seṅ ge khyod la phyag ’tshal lo |

| rtogs pa brjod pa’i rjen śul che | | dam chos spo la brten pa mdzad || chos gźan thams cad
’jig pa po | | glaṅ chen khyod la phyag ’tshal lo |

| chos ñid śin tu dka’ ba’i lam | | khyod min gźan gyis mi bzod pa || gsal mdzad gśegs pa ’jig
rten gyi | | lam gyur khyod la phyag ’tshal lo |

| des la śin tu dul gyur bzod | | śugs daṅ mdog bzaṅ stobs daṅ ldan || yan lag rdzogs pa’i sku
mṅa’ ba | | caṅ mkhyen khyod la phyag ’tshal lo |

| phyogs daṅ ’bab stegs spyod yul mkhyen | | brtan la gya gyur bźud mi mṅa’ || khyu mchog gnas
ni brñes pa yi | | khyu mchog khyod la phyag ’tshal lo |

| gdul ba’i thabs ni mtha’ dag la | | mkhas śiṅ le lo mi mṅa’ ba || skyes bu ’dul ba’i kha lo
sgyur | | mchog rab khyod la phyag ’tshal lo |

| gaṅ slad khyod bstan ma mchis na | | skal ba can yaṅ don ma mchis || de slad kun gyi tshaṅs
spyod du | | gyur pa khyod la phyag ’tshal lo |

| yon tan phun sum ldan pa kun | | yaṅ dag śin tu yoṅs rdzogs pas || dge ba’i chos la ’dun pa
kun | | rgyun bcad khyod la phyag ’tshal lo |

| ñid kyi don ni yoṅs rdzogs śiṅ | | gźan gyi don la mṅon brtson pa || dam chos bdud rtsi stsol
ba po | | tshim par mdzad pa khyod la ’dud |

| ’jigs pa du mas ’jigs gyur pa | | pha rol min pa’i mu spaṅs nas || chu bo chen po legs brgal
źiṅ | | pha rol gśegs pa khyod la ’dud |

| ñon moṅs nad ni kun sel ciṅ | | zug rṅu thams cad ’byin mdzad pa || rig sṅags ’chaṅ ba grub
pa po | | sman pa’i gtso bo khyod la ’dud |

| mchog daṅ mchog min lam mkhyen ciṅ | | źi ba’i phyogs ni ston par mdzad || grub daṅ bde ba
bskyed pa yi | | ded dpon khyod la phyag ’tshal lo |

| rtag tu mun pa sel mdzad ciṅ | | rtag tu spyan ni ’byed par mdzad || rtag tu mi bsñel chos
mṅa’ ba | | śin tu bźeṅs pa khyod la ’dud |

| dri ma’i ’dam ni ṅes bkrus pas | | dam chos dri ma med pa’i chu || mchog tu rnam par byaṅ
ba po | | śin tu rnam dag khyod la ’dud |

| sdig daṅ bsod nams spaṅs gyur ciṅ | | stoṅ pa ñid kyis bsgos pa’i bdag || phan daṅ gnod pa
bgyid sñom pa | | śin tu chags bral khyod la ’dud |

| kun nas ñon moṅs bag chags kun | | ṅes par bsal bas dri med thugs || rnam thar sgrib pa mi
mṅa’ ba | | śin tu rnam grol khyod la ’dud |

| blta bas mi ṅoms gzugs mṅa’ źiṅ | | sku ni ’od kyis lhag ger mdzes || blta bya kun gyi naṅ
na ni | | blta na sdug mchog khyod la ’dud |
